Инструкция

Инструкция по установке и обновлению JViewer

Внимание

Работа на 32-битных системах не гарантируется

Установка

  1. Загрузите архив для вашей системы и распакуйте его
  2. Перейдите в папку jviewer
  3. Выполните файл install_update для загрузки и сборки приложения, и дождитесь окончания его работы
  4. Выполните файл deploy для развертывания приложения на сервер
  5. Выполните файл start для запуска сервера

Обновление

  1. Перейдите в папку jviewer и остановите сервер, запустив файл stop
  2. Выполните файл install_update, для обновления и пересборки приложения
  3. Выполните файл deploy для развертывания приложения на сервер
  4. Вновь запустите сервер, выполнив файл start
