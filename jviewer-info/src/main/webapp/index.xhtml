Java Viewer Application

Онлайн-площадка для проведения веб-конференций по HTML, JavaScript, CSS.

Основной функционал:

  • Регистрация
  • Вход в систему
  • Создание отдельных комнат для обучения
  • Демонстрация кода в реальном времени
  • Интерпретация введенного кода и демонстрация полученного результата

Используемые технологии:

  • Java 8
  • Spring 4
  • Spring Security 3.2
  • JSF 2
  • Primefaces 5
  • SQLite
  • Maven

Дополнительно:

  • Tomcat 8
  • Sonar
